users_courses__certificate_payments

Description

A summary table that provides details on whether an enrollment has met the payment condition for the eligibility of earning a Course Certificate.

Columns

NameDescription
taiwan_user_idEncrypted Coursera user id for taiwan data.
course_id22 character unique string identifying an individual course
met_payment_conditionWhether the payment condition for a Course Certificate was met, by the logical OR of the three paths below.
was_paymentA user made a "single payment" for the course or a "bulk payment" for a specialization that contains the course. This payment was not refunded.
was_finaid_grantA user applied and was granted financial aid for the course.
was_group_sponsoredA user was sponsored to meet the payment criteria by being a member of a Coursera group.

SQL create statement

CREATE TABLE users_courses__certificate_payments (
    taiwan_user_id VARCHAR(50) NOT NULL
    ,course_id varchar 
    ,met_payment_condition bool 
    ,was_payment bool 
    ,was_finaid_grant bool 
    ,was_group_sponsored bool 
);
