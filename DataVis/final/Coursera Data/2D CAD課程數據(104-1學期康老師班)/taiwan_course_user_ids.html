taiwan_course_user_ids

Description

Encrypted user id mapping table. Use this table to connect learner data from exports, which identifies taiwan learners using taiwan_user_id, to course-scoped data from external surveys.

Columns

NameDescription
taiwan_user_idEncrypted Coursera user id for taiwan data.
_2d_cad_user_idEncrypted Coursera user id for 2d-cad data.

SQL create statement

CREATE TABLE taiwan_course_user_ids (
    taiwan_user_id VARCHAR(50) NOT NULL
    ,_2d_cad_user_id VARCHAR(50) NOT NULL
    ,PRIMARY KEY (taiwan_user_id)
    ,FOREIGN KEY (taiwan_user_id) REFERENCES users(taiwan_user_id)
);
