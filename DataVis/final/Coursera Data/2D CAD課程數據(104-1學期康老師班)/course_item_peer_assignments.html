course_item_peer_assignments

Description

For each course, provide the mapping between course items and peer assignments.

Columns

NameDescription
course_id22 character unique string identifying an individual course
course_item_id5 character alphanumeric string identifying an individual item within a course. course_item_id need not be unique, but a course_item_id/course_id pair is unique.
peer_assignment_idThis is a 'course_content_id', and is versioned. (e.g ABC@1 is version 1)

SQL create statement

CREATE TABLE course_item_peer_assignments (
    course_id varchar 
    ,course_item_id varchar 
    ,peer_assignment_id varchar 
);
