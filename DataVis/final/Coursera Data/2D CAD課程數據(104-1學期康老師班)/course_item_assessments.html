course_item_assessments

Description

For each course, provide the mapping between course items and assessments.

Columns

NameDescription
course_id22 character unique string identifying an individual course
course_item_id5 character alphanumeric string identifying an individual item within a course. course_item_id need not be unique, but a course_item_id/course_id pair is unique.
assessment_idEach quiz is assigned a unique id that is a combination of a base id and version.

SQL create statement

CREATE TABLE course_item_assessments (
    course_id varchar 
    ,course_item_id varchar 
    ,assessment_id varchar 
);
